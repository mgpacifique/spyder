Now that we have managed our variables of interest and visualized their relationships graphically, we are ready to test those relationships statistically. The first step is an Analysis of Variance (ANOVA), which assesses whether the means of a quantitative response variable are statistically different across the categories of an explanatory variable.

Running the ANOVA

To test my theories regarding Martian crater morphology, I used the quantitative variable DEPTH_RIMFLOOR_TOPOG (Crater Depth) and the categorical variable diameter_category (Crater Size Category: Small, Medium, Large, Basin). Our null hypothesis is that there is no difference in the mean depth across size categories, while the alternative hypothesis is that there is a significant difference.

marscrater.ipynb
Python 
import statsmodels.formula.api as smf
import statsmodels.stats.multicomp as multi

# Subset data for ANOVA, dropping missing values
sub1 = clean_data[['DEPTH_RIMFLOOR_TOPOG', 'diameter_category']].dropna()

# Using ols function for calculating the F-statistic and associated p-value
model1 = smf.ols(formula='DEPTH_RIMFLOOR_TOPOG ~ C(diameter_category)', data=sub1)
results1 = model1.fit()
print(results1.summary())

ANOVA Results

Running the OLS regression model on our sample of 76,804 valid craters yielded the following primary statistical results:

Statistic Value
F-statistic 7035.0
Prob (F-statistic) 0.00

Because the Prob (F-statistic) is 0.00 (which is less than the standard 0.05 threshold), our ANOVA is statistically significant. We can confidently reject the null hypothesis and state that there is a statistically significant difference in the mean crater depth across the different size categories.

Post Hoc Comparisons

Because our categorical explanatory variable has more than two levels, a significant ANOVA only tells us that at least one category mean is different, but it doesn't specify which ones. To find out, we must run a post hoc test using Tukey's HSD (Honestly Significant Difference).

marscrater.ipynb
Python 
# Run Tukey's HSD to evaluate all pairwise comparisons
mc1 = multi.MultiComparison(sub1['DEPTH_RIMFLOOR_TOPOG'], sub1['diameter_category'])
res1 = mc1.tukeyhsd()
print(res1.summary())

Interpretation: Confirming the Plateau Theory


The post hoc analysis shows a reject = True and an adjusted p-value of 0.0 for every single pairwise comparison. This means that the mean crater depth is statistically significantly different between every size category.


However, when we analyze the magnitude of these differences (the Mean Diff), it confirms the structural theory we derived from our earlier bivariate visualizations. For instance, the difference in depth between Small and Large craters is substantial (-0.8394 km). Yet, when moving from a Large crater to a massive Basin structure, the depth only increases by -0.2064 km. This statistical evidence perfectly aligns with the geological "plateau" theory: while impacts initially excavate deeper, the maximum depth of the largest craters is inherently constrained by crustal rebound and long-term geological infilling.
