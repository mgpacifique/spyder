Chi-Square Test of Independence

Following our analysis of variance (ANOVA) on crater depth, we turn to categorical data. We will use a Chi-Square Test of Independence to determine if there is a statistically significant association between the size of a crater and whether it produces a Radial ('Rd') ejecta blanket.

Running the Chi-Square Test

We use diameter_category (Small, Medium, Large, Basin) as our explanatory variable and a newly created binary response variable is_Rd, which is 1 if the crater morphology is 'Rd' (Radial) and 0 otherwise.

marscrater.ipynb
Python 
import scipy.stats

# Create a contingency table (crosstab)
ct = pd.crosstab(ours['diameter_category'], ours['is_Rd'])

# Run the Chi-Square test
chi2, p, dof, expected = scipy.stats.chi2_contingency(ct)
print(f"Chi-Square Statistic: {chi2}")
print(f"p-value: {p}")

Chi-Square Results

The overall Chi-Square test yielded a massive statistic (21,259.13) and a p-value of effectively 0.0 (well below the 0.05 threshold). We confidently reject the null hypothesis: there is a statistically significant association between crater size and the presence of Radial ejecta morphology.

Size Category Expected 'Rd' Observed 'Rd' % with 'Rd' Morphology
Small 21,816 14,570 ~4.3%
Medium 2,941 9,661 ~21.2%
Large 127 647 ~32.9%

By comparing expected and observed frequencies, we can see that Small craters have Radial ejecta significantly less often than expected by chance. Conversely, Medium and Large craters produce Radial ejecta significantly more often than expected.

Post Hoc Paired Comparisons

Because the overall test only tells us that at least one group is different, we run post hoc paired comparisons to identify exactly where the differences lie.

A Note on the Basin Category


In the overall contingency table, the "Basin" category only contained 14 craters with Radial ejecta (and an expected frequency of ~6.7). Chi-Square tests become unreliable when expected cell frequencies drop too low (typically below 5). To ensure statistical rigor and avoid spurious results, we restricted our post hoc analysis to only the Small, Medium, and Large categories.

Testing 3 pairs means we must apply a Bonferroni adjustment to our significance level to prevent false positives: 0.05 / 3 = 0.0167.

marscrater.ipynb
Python 
from itertools import combinations

# Restrict analysis to exclude the 'Basin' category due to low counts
valid_categories = ['Small', 'Medium', 'Large']
posthoc_data = ours[ours['diameter_category'].isin(valid_categories)].copy()

# Calculate the Bonferroni adjusted significance level
adjusted_alpha = 0.05 / 3

# Generate all pairwise combinations of our categories
pairs = list(combinations(valid_categories, 2))

# Loop through each pair, subset the data, and run the Chi-Square test
for group1, group2 in pairs:
    subset = posthoc_data[(posthoc_data['diameter_category'] == group1) | 
                          (posthoc_data['diameter_category'] == group2)]
    
    ct_pair = pd.crosstab(subset['diameter_category'], subset['is_Rd'])
    chi2, p, dof, expected = scipy.stats.chi2_contingency(ct_pair)
    print(f"--- {group1} vs. {group2} ---")
    print(f"p-value: {p}")

Geological Interpretation


The post-hoc analysis shows a reject = True for every single pairwise comparison. The proportion of craters with Radial ejecta is statistically significantly different between every size category.


The statistical analysis reveals a strong, positive association between the size of a Martian crater and the likelihood of it forming a Radial ('Rd') ejecta morphology. As craters increase in size from Small to Medium to Large, the proportion of Radial ejecta blankets increases significantly at every step. Geologically, this suggests that the energy of the impact, or the depth of the subsurface material being excavated by larger impactors, plays a crucial role in forming these distinct, sweeping radial patterns, whereas smaller, lower-energy impacts generally fail to produce them.
